Big MVP Update

We are excited to announce that we are officially putting out our first version of WISSH for the public to use and contribute to! Check out some of the great features WISSH has so far:

Secure Web-Based CLI

To create a web application that offers zero compromises, we strove to implement a secure command-line interface (CLI) accessible within the app. With the help of xterm.js, we built a CLI that uses a secure web socket to communicate directly with the server. The terminal acts exactly like a pseudoterminal thanks to node-pty, so you can use programs exactly as you would on your personal machine through the CLI.

Docker Deployable Server

With Docker, you can use the docker compose command to start and stop WISSH. We recommend viewing our README.md for instructions on how to run WISSH.

User Interface

Navigation Bar

We added a navigation bar feature to our React app. Though it does not change much between the two routes, we are working on making it route to other pages. Stay tuned for the updates to come!

Prototype File Structure

As of now, we are working on a functioning interface of the user's files. This makes accessing files easier and more user-friendly than the command line. Currently, we have in place a prototype image of what we would like the file graphics to look like. We plan to connect the GUI with the terminal in order to create a seamless experience.

Don't forget to go to our GitHub to find out how you can use WISSH today!
