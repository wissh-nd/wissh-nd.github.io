WISSH is Open Source

January 31, 2024

A web-based SSH client or WISSH, is an application that allows users to access Secure Shell (SSH) servers through a web browser. This application would allow students to access the student machines without needing to use the command line on a computer. Users would be able to view/edit files from their phones, download/upload files/folders without needing to manually use the SFTP protocol, and execute programs.
